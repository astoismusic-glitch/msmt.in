# AstroManish Website

Dark cinematic static website for AstroManish, built with HTML, CSS, and JavaScript.

## Preview Locally

```powershell
node server.js
```

Open:

```text
http://localhost:4173
```

Current PowerShell preview server:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\tcp-server.ps1
```

Open:

```text
http://localhost:4176
```

## Publish On GitHub Pages

Upload the full folder contents to a GitHub repository, then enable GitHub Pages from the repository settings. Use the main branch and root folder.

Required files:

- `index.html`
- `styles.css`
- `script.js`
- `server.js`
- `assets/`
